# DriftLoom 2.0 Vision

## Product Positioning

DriftLoom is a private idea and reflection app that helps people capture scattered thoughts, then discover meaningful patterns across notes, memories, projects, moods, and creative fragments.

DriftLoom is not a dream app. AuraLunis owns dreams, lucid practice, symbols, and moon cycles. DriftLoom owns private thinking, idea capture, project shaping, and personal pattern discovery.

## Core Promise

Most note apps help users capture.

DriftLoom helps users notice.

The magic moment is when someone realizes:

> I forgot I wrote that, and now I see how it connects.

## Tagline Options

- Capture the fragments. Find the pattern.
- Capture ideas. Discover connections.
- Where scattered thoughts become direction.

---

# 1. Pattern DNA

## Purpose

Pattern DNA is DriftLoom's signature insight system. It turns fragments into a living profile of what the user keeps returning to.

## Inputs

- Tags
- Topics
- Mood
- Capture type
- Projects
- Favorites
- Revisit queue
- Time and date
- Frequency over time

## Outputs

- Top recurring topics
- Top moods
- Active idea categories
- Weekly changes
- Pattern archetype
- Suggested next focus

## Example Output

> Your Pattern DNA this week is 42% Builder, 31% Explorer, and 27% Reflector.
>
> Business, product ideas, and pricing are your strongest recurring themes.

## UI Placement

- Home: small Pattern DNA summary card
- Insights/Loom: full Pattern DNA section
- Settings/Plus: Pattern DNA+ as premium unlock

# 2. Weekly Reflection

## Purpose

A weekly recap that helps the user understand what they captured and what deserves attention next.

## Sections

- Top themes this week
- Most used tags
- Mood shifts
- Best idea captured
- Unfinished thought worth revisiting
- Suggested next step

## Example Output

> This week, your notes centered on app ideas, pricing, and creative direction.
>
> Your strongest unfinished thread is DriftLoom positioning. Consider turning it into a project brief.

## UI Placement

- Home: Sunday Weekly Reflection card
- Vault: saved weekly reports
- Plus: advanced weekly report history

---

# 3. Future Self

## Purpose

Future Self lets users write notes that unlock later, creating emotional long-term value and reflection moments.

## Core Actions

- Open in 7 days
- Open in 30 days
- Open in 90 days
- Open on custom date
- Save to Time Capsule

## Example Use Cases

- A letter to future me
- A project prediction
- A private memory
- A goal check-in
- A note after a major decision

## UI Placement

- Capture: Future Self option
- Vault: Time Capsule section
- Home: upcoming Future Self card

---

# 4. Relationship Map

## Purpose

Relationship Map visually connects related fragments, ideas, tags, and projects. This is the premium feature that separates DriftLoom from ordinary notes apps.

## Inputs

- Shared tags
- Shared words
- Same project
- Same mood
- Same time period
- User favorites
- Manual links

## Outputs

- Node map
- Topic clusters
- Project clusters
- Hidden connections
- Suggested links

## Example Output

Business
├─ DriftLoom
├─ Pricing
├─ App Store
└─ Revenue

## UI Placement

- Loom: interactive map
- Projects: project relationship view
- Plus: unlimited maps and advanced clustering

---

# 5. Insight Score

## Purpose

Insight Score gives users a simple sense of momentum without turning DriftLoom into a gamified productivity tracker.

## Score Inputs

- Capture consistency
- Revisits
- Projects created
- Reflections completed
- Notes connected
- Weekly review completed

## Example Output

> Insight Score: 78
>
> You are capturing consistently and revisiting older ideas more often. Your next best move is to connect three related fragments into a project.

## UI Placement

- Home: primary score card
- Insights/Loom: breakdown
- Settings: explanation and privacy note
