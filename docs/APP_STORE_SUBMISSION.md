## App Review Notes (for Apple reviewer)
This app uses the Anthropic Claude API for AI-powered dream interpretation features. AI features are gated behind a subscription paywall with a 7-day free trial. 

All core journaling features are fully functional without a subscription:
- Dream logging with all 39+ data fields
- Moon phase calculator
- Dream Tarot
- Sleep Cycles education
- Lucid dreaming guides
- Dream Dictionary
- Themes and customization

Premium (subscription) features:
- AI Dream Interpretation
- 6-mode AI Analysis
- AI Bedtime Story
- AI Affirmations  
- AI Title Generator

The app does not collect personal data beyond what the user voluntarily enters in their dream journal. All data is stored locally on device via AsyncStorage. AI features send dream text to Anthropic's API for processing — no data is stored on our servers.

Demo account: Not required — the app works immediately upon download.
In-app purchases: Auto-renewable subscriptions ($3.99/month, $29.99/year) with 7-day free trial.
