# 🌀 Driftloom — Turn Creative Drift into Direction

> Capture stray ideas, voice memos, and screenshots. Driftloom finds patterns in your fragments and turns them into buildable project directions.

## Tech Stack
- **React Native** (Expo) with TypeScript
- **RevenueCat** for subscriptions
- **AsyncStorage** for local persistence
- **Expo AV** for voice recording
- **Expo Image Picker** for camera/photos

## Quick Start
```bash
npm install
npx expo start --dev-client
```

## Structure
```
App.tsx              # Full app (6 tabs, 65KB)
```

## 6 Tabs
- **Home** — Streak, insights, momentum, daily quests
- **Capture** — Text, voice, camera, links, location fragments
- **Loom** — Pattern detection, signals, synthesis
- **Projects** — Incubated ideas with build plans
- **Vault** — Favorites, archived, search & filter
- **Settings** — Theme, notifications, export, subscription
